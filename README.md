### How to run the code

1. Clone this repository

2. Install packages
    ```sh
    yarn install
    ```
3. Connecting to postgres SQL

    ```sh
    - setting file database.js
    - yarn db:create; yarn db:migrate; yarn db:seed
    ```

4. Run the code

    ```sh
    yarn develop
    ```
